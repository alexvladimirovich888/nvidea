API Documentation

Quick Start

agent.init(mode="edge", gpu=0);
agent.load_model("x402/inference-v4.02");
agent.deploy("production");

1. Flash firmware to compatible NVIDIA hardware
2. Configure network and authentication keys
3. Initialize with agent.bootstrap() command
